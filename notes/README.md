## 怎么用这些笔记

- 现在不要试图一次背完它们。
- 每天只拿和当天 demo 有关的一小段出来复习。
- 如果某个概念总是记不住，就把它改写成 `Q: ... / A: ...` 放到对应 demo 的 README。
- 适合视觉化的内容，复盘时整理成脑图、流程图或学习卡片。
