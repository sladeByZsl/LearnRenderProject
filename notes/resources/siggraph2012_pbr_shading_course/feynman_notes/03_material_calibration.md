本节只抓一个核心问题：

```text
为什么 PBR 项目里，材质参数不能只靠“当前场景看起来好看”来调？
```

## 一句话答案

PBR 项目需要材质校准，因为材质参数应该描述材料本身，而不是偷偷补偿某个场景里的灯光、曝光或后处理。

## 讲给 12 岁小孩听

假设你要给一个玩具上颜色。

如果你只在昏暗房间里看它，你可能会把颜色涂得特别亮。可是拿到太阳底下，它就会亮得不正常。

材质校准就像是：我们先找一个标准房间、标准灯光、标准参照物，再决定这个玩具到底应该是什么颜色、是不是金属、表面有多粗糙。

这样以后把它放到森林、沙滩、夜晚、室内，它都不会离谱。

## 为什么需要“合理范围”

PBR 不是说美术不能调，而是说参数最好有边界。

比如：

```text
albedo/base color：应该是物体本色，不应该把阴影和高光画进去
metallic：大多数真实材料要么是金属，要么不是金属，不常停在中间
roughness：控制反射模糊程度，不应该拿来乱补亮度
ao：主要表示缝隙和角落里的环境光遮蔽，不应该当成普通阴影贴图乱乘
```

如果参数乱调，短期看可能“这个镜头好看了”，但长期会出问题：

- 换光照就坏。
- 换关卡就坏。
- 不同美术做出来的材质风格不一致。
- Debug 时不知道到底是灯光错、贴图错，还是 shader 错。

## 材质校准到底在校准什么

可以理解成校准三件事：

### 1. 校准本色

确认 albedo/base color 是材料自己的颜色。

不要把这些东西画进去：

```text
阴影
高光
环境光颜色
AO 过重的黑边
摄影曝光补偿
```

因为这些应该由光照、环境、渲染管线来产生。

### 2. 校准反射

确认 metallic / specular / roughness 描述的是材料反射习惯。

比如：

- 光滑金属应该有清晰反射。
- 粗糙金属应该有模糊反射。
- 塑料有高光，但高光颜色通常不等于塑料本色。

### 3. 校准观察环境

调材质时要有稳定的参考环境。

这就像 Unity 里做一个材质球测试场景：

```text
固定灯光
固定曝光
固定 skybox / reflection probe
放几个标准球作参考
```

否则你不知道材质变化是因为贴图改了，还是因为环境变了。

## 和 Unity 的对应关系

在 Unity 里，可以这样做一个最小材质校准场景：

```text
一个中灰背景
一个 Directional Light
一个稳定的 HDRI / Skybox
一排材质球
一个白色非金属球
一个黑色非金属球
一个金属球
一个粗糙球
一个光滑球
```

每做一个新材质，都先放到这个场景里看：

- base color 是否像真实材料？
- metallic 是否只有金属才开？
- roughness 是否符合表面状态？
- 换一个环境后是否仍然可信？

第一节说：

```text
PBR 是为了让材质更稳定。
```

第二节说：

```text
材质的本质是处理光。
```

第三节补上生产规则：

```text
如果材质参数不校准，就会把光照问题、贴图问题、材质问题混在一起。
```

所以 PBR 不只是 shader 公式，它也是一套资产制作纪律。

## 本节容易误解的地方

### 误解 1：材质校准会限制美术发挥

不完全对。校准限制的是“乱补偿”，不是限制风格。你仍然可以做写实、卡通、夸张风格，只是要知道哪些变化是材质本身，哪些变化是灯光、后处理、风格化 shader。

### 误解 2：只要用了 metallic/roughness 工作流，就是 PBR

不够。用了这些贴图只是开始。如果 albedo 里画了很多阴影、高光，roughness 被拿来乱补亮度，结果仍然可能不稳定。

### 误解 3：材质在当前镜头好看就行

短片或单镜头也许可以，但游戏项目里资产会被复用。材质需要在很多场景、时间、天气、光照条件下都说得过去。

## 你今天只需要记住

```text
PBR 材质参数应该描述材料本身，不应该偷偷替灯光和后处理擦屁股。
```

## 30 秒复述版

材质校准是为了让 PBR 参数保持稳定含义。albedo 应该是物体本色，不要画进阴影和高光；metallic 应该表达是不是金属；roughness 应该表达反射模糊程度，而不是乱当亮度旋钮。调材质时最好放在统一测试环境里看，这样才能知道问题来自材质、灯光还是 shader。

## Q&A

### Q：材质校准和 Unity 里的材质球预览有什么关系？

A：材质球预览就是一个很小的校准环境。它让你在相对固定的灯光和反射环境下观察材质，减少场景因素干扰。但正式项目里通常还会做更明确的材质测试场景，方便比较不同材质是否统一。

### Q：为什么说 albedo 不能包含光照？

A：因为 albedo 是物体本色。如果把阴影、高光、环境色画进 albedo，等真实灯光再照一次时，相当于把光照效果叠加了两遍。这样换光照环境时就很容易不自然。
