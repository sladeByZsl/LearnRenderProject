# 02 光遇到表面后发生了什么：费曼学习笔记

本节只抓一个核心问题：

```text
光打到一个物体表面后，为什么会产生不同材质的视觉差异？
```

## 一句话答案

光打到物体后，不是简单地“变成颜色”，而是会被表面反射、进入物体、在内部散射、被吸收，最后剩下进入眼睛的那部分才形成我们看到的材质。

## 讲给 12 岁小孩听

想象你拿手电筒照一个东西。

如果照到镜子，光大部分会被弹回来，所以你能看到清楚反光。

如果照到红色橡皮，光会钻进橡皮表面一点点，有些颜色被吃掉，红色那部分又散出来，所以你看到它是红的。

如果照到粗糙石头，光会往很多乱七八糟的方向散开，所以它没有很亮很清楚的高光。

PBR 就是在问：这个材料会怎么处理照到它身上的光？

## 三条最重要的路

### 1. 表面反射

光刚碰到表面，就有一部分被弹回来。

这部分主要影响：

```text
高光
反射
Fresnel
金属感
```

在 Unity 里，你会把它直觉对应到：

```text
Smoothness / Roughness
Metallic
Specular
Reflection Probe / Skybox 反射
```

### 2. 进入物体内部

对于非金属，光通常会进入表面下面一点点。

进去以后，它可能会：

```text
被吸收一部分
在内部散射
再从附近的位置跑出来
```

这部分更接近我们平时说的“物体本色”。

比如红色塑料不是因为它把所有光都变红了，而是因为它吸收了很多非红色光，剩下红色比较多地回到眼睛里。

### 3. 被吸收

有些光进了材料之后就没出来。

这就是为什么东西会有颜色、会变暗、会有材质差异。

如果一个物体吸收了大部分可见光，它就接近黑色。

## 金属和非金属的关键区别

先记一个很有用的简化版：

```text
非金属：有明显 diffuse，本色主要来自内部散射后出来的光。
金属：光很难进入内部，本色更多体现在表面反射里。
```

所以在 PBR 里：

- 塑料、木头、石头这类非金属，通常有明显 base color / diffuse。
- 金、铜、铁这类金属，颜色更多出现在 specular reflection 里。

这也是为什么 metallic 不是“让东西更亮”的按钮，而是在改变光和材质的分工方式。

## roughness 在这里怎么理解

roughness 不是说物体轮廓真的变坑坑洼洼。

它说的是：在非常微小的尺度上，表面像不像很多小镜子组成。

```text
roughness 低：小镜子方向比较整齐，反射清楚，高光小而亮。
roughness 高：小镜子方向很乱，反射模糊，高光大而散。
```

讲给小孩听可以这样说：

```text
光滑桌面像整齐的小镜子，能照出亮亮的反光。
磨砂桌面像歪七扭八的小镜子，光被弹得到处都是，所以反光糊掉了。
```

## 这里和第一节怎么连起来

第一节说：PBR 是为了让材质更稳定。

这一节解释：为什么材质能被更稳定地描述。

因为我们不再只是说“这个像素调亮一点”，而是在描述：

```text
光有多少被表面反射？
有多少进入材料？
有多少被吸收？
表面微小结构有多乱？
```

这些问题比“这个场景里看起来好不好看”更接近材料本身。

## 本节容易误解的地方

### 误解 1：diffuse 就是颜色

不准确。diffuse 更像是“光进入非金属材料后，散射出来的那部分”。颜色通常来自材料对不同波长光的吸收和散射结果。

### 误解 2：金属没有颜色

不对。金属有颜色，但它的颜色更多体现在反射光里。比如金子反射出来偏黄，铜反射出来偏红。

### 误解 3：roughness 越高，物体越暗

不一定。roughness 主要改变反射分布：高光从小而集中变成大而分散。它可能看起来没那么刺眼，但不是简单的“变暗旋钮”。

### 误解 4：PBR 参数都是美术随便调的

不对。PBR 允许艺术控制，但参数最好仍然落在合理范围里。否则材质换一个光照环境就容易崩。

## 你今天只需要记住

```text
材质不是直接“拥有颜色”，而是用自己的方式处理照到它身上的光。
```

## 30 秒复述版

光打到物体后，一部分会直接从表面反射，一部分会进入物体内部，被吸收或散射后再出来。非金属的本色通常来自内部散射后的光，金属的颜色更多体现在表面反射里。roughness 描述的是微小表面有多乱，越光滑反射越清楚，越粗糙反射越模糊。PBR 的材质参数，本质上是在描述材料如何处理光。

## 费曼检查

请你试着回答：

1. 为什么红色塑料看起来是红的？
2. 为什么金属和非金属不能只靠“颜色不同”来区分？

## Q&A

### Q：这一节是不是要开始背公式？

A：先不要。现在只需要理解“光被反射、进入、散射、吸收”这个过程。公式后面只是为了把这些过程变成 GPU 能计算的近似。

### Q：为什么说 metallic 改变的是光的分工？

A：因为非金属通常有明显 diffuse，本色主要在漫反射里；金属的 diffuse 很弱，本色更多进入 specular 反射里。所以 metallic 不是简单调亮或调黑，而是在改变材质如何分配 diffuse 和 specular。

### Q：为什么粗糙度越高，镜面反射形状反而更大？

A：这里的“大”不是说反射更清楚，而是说高光被摊开了。光滑表面像一堆方向很整齐的小镜子，光会集中弹向一个方向，所以高光小、亮、锐利。粗糙表面像一堆方向很乱的小镜子，光会被弹到更多方向，所以高光范围变大、边缘变糊、峰值没那么亮。可以记成：光滑是“小而亮”，粗糙是“大而散”。
